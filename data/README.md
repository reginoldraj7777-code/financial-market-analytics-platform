# Data Folder

This folder is intended to store raw and processed stock market datasets used for time-series analysis and dashboard visualization.

Due to size and licensing constraints, the dataset is not included in this public repository.

## Dataset Description

The project uses historical stock price data containing:

- Open, High, Low, Close prices
- Volume
- EPS values
- Beta ratios
- Derived technical indicators (moving averages, volatility metrics)

## How to Recreate

You can recreate the dataset by:

- Downloading historical stock data from Yahoo Finance / Kaggle
- Importing into MySQL tables
- Running the preprocessing queries provided in the notebooks and src scripts
